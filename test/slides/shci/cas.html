Complete Active Space (CAS)

  • We can perform HCI on a subset of the orbitals, called the active orbitals
  • Complete active space (CAS) methods
  • CAS(3e,5o) means correlating 3 electrons in 5 orbitals
  • Cost scales exponentially with the size of the active space and not the system
  • Now we can treat much larger systems
  • We can (and will in this work) optimize orbitals for the set of determinants
  • So rather than doing FCI on all the orbitals we can perform it on a subset.
  • We start by dividing all of our orbitals into three categories: inactive, which are always doubly occupied, virtual, which are always unoccupied, and active which can have any occupation between 0 and 2.
  • The active orbitals here are the important ones.
